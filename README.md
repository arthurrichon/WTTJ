## Installation

After cloning:
* Si la version de npm < 5.2 -> `npm install npm@latest-g`
* `npm install && npm start`
* Got to `localhost:3000/col=X&line=Y` -- Replace X & Y by number, ex: `localhost:3000/col=3&line=2`
* Sample data is stored in `data.json`
